Color Usage Guidelines

Accessibility Standards

  • All color combinations meet WCAG 2.1 AA standards
  • Minimum contrast ratio of 4.5:1 for body text
  • Minimum contrast ratio of 3:1 for large text
  • Color is never the sole indicator of meaning

Application Rules

  • Follow 60-30-10 rule: 60% neutral, 30% primary, 10% accent
  • Ocean Blue should dominate in brand applications
  • Use accent colors sparingly for maximum impact
  • Maintain consistency across all touchpoints

Typography Best Practices

  • Hierarchy: Maintain consistent scale relationships across all materials
  • Line Height: Use 1.6 ratio for body text, 1.2-1.4 for headings
  • Spacing: Apply consistent bottom margins between elements
  • Contrast: Ensure sufficient color contrast for all text

Technical Specifications

  • Web: Use rem units for scalability and accessibility
  • Print: Convert to pt sizes with appropriate leading
  • Minimum: 12px/14pt for body text, 16px/18pt preferred
  • Mobile: Increase base size to 16px minimum for readability

Implementation Guidelines

Comprehensive standards for maintaining brand consistency across all applications

Logo Implementation

  • ✓ Do: Maintain aspect ratio at all sizes
  • ✓ Do: Use approved color versions only
  • ✓ Do: Respect minimum clear space requirements
  • ✗ Don't: Stretch, rotate, or skew the logo
  • ✗ Don't: Change colors or add effects
  • ✗ Don't: Place on busy or low-contrast backgrounds

Color Application

  • ✓ Do: Use Ocean Blue as primary brand color
  • ✓ Do: Follow 60-30-10 color distribution rule
  • ✓ Do: Ensure WCAG accessibility compliance
  • ✗ Don't: Use colors outside the approved palette
  • ✗ Don't: Rely solely on color to convey information
  • ✗ Don't: Use low contrast color combinations

Typography Rules

  • ✓ Do: Use Roboto for all brand applications
  • ✓ Do: Maintain consistent hierarchy and spacing
  • ✓ Do: Use appropriate font weights for emphasis
  • ✗ Don't: Mix multiple typeface families
  • ✗ Don't: Use decorative or script fonts
  • ✗ Don't: Set body text smaller than 14px/12pt
